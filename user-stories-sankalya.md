# User stories 

**Researcher:** Sankalya Tharindi  
**Date:** 2026-03-09
**Branch:** `doc/sankalya-research`

## 1. Passenger

- As a passenger I need to view the bus routes in a map
- As a passenger I need to view the interior design of the bus with the available seat count
- As a passenger I need to view the deatails of the bus, driver
- As a passenger I should be able to buy a ticket through the system
- As a passenger I should be able report issues through the system
- As a passenger I should get updates about delays, traffic information through the system

## 2. Admin

- As an admin I need to view the total number of passengers registered with the details
- As an admin I need to view the details of the buses
- As an admin I need to view the passenger count
- As an admin I should be able to get the payment summary
- As an admin I need to get the passenger reported issues and reply to them

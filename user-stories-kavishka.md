# Metro System User Stories & Requirements
**Author:** Kavishka Rasanjana Jayathissa  
**Status:** Draft / Research Phase  

---

## 1. The Daily Passenger (Commuter)
*Focus: Speed, clarity, and stopping the "change money" arguments.*

* **Real-time Tracking:** As a passenger, I want to see exactly where my bus or train is on a live map so I don't have to stand at the halt for an hour wondering if it's coming. (**High Priority**)
* **Fare Calculator:** As a passenger, I want to enter my start and end stages to see the exact price (e.g., Rs. 34.00) before I board, so I can have the exact change ready. (**High Priority**)
* **Language Options:** As a Sri Lankan user, I want to switch the entire site between Sinhala, Tamil, and English instantly so I don't get confused by technical terms. (**High Priority**)
* **Station Guide:** As a passenger, I want to know if a station has a clean washroom, an ATM, or a nearby landmark so I can plan my stops better. (**Medium Priority**)
* **Delay Alerts:** As a passenger, I want a quick "Alerts" section on the homepage to see if a specific route is blocked by traffic or a breakdown. (**Medium Priority**)

---

## 2. The System Admin (The Controller)
*Focus: Keeping the data accurate and the system running smoothly.*

* **Fare Management:** As an admin, I need a simple dashboard to update the price-per-stage whenever the government changes fuel prices, so the website stays accurate. (**High Priority**)
* **Route Control:** As an admin, I want to add new routes or temporary "shuttle" paths for special events (like Poya days) without touching the code. (**High Priority**)
* **Driver Assignment:** As an admin, I need to see which driver/conductor is logged into which vehicle so I can monitor who is actually on the road. (**Medium Priority**)
* **Passenger Feedback:** As an admin, I want to see a log of complaints or suggestions sent by users so we can improve specific bus routes. (**Low Priority**)

---

## 3. The Driver or Conductor (The Provider)
*Focus: Providing data without it being a distraction.*

* **Trip Toggle:** As a driver, I want a big "Start Trip" and "End Trip" button on my phone so my location only shows on the public map when I am actually working. (**High Priority**)
* **Incident Reporting:** As a conductor, I want a 2-click way to report a "Breakdown" or "Heavy Traffic" so that passengers waiting at the next halts are automatically notified. (**Medium Priority**)

---

## 4. The Tourist (The Visitor)
*Focus: Making the system "tourist-friendly" like Japan’s Metro.*

* **Digital Passes:** As a tourist, I want to see how to get a "Day Pass" or "Weekly Card" so I don't have to figure out local cash every single time I travel. (**Medium Priority**)
* **Landmark Mapping:** As a tourist, I want the system to suggest which station I should get down at to reach popular places like Galle Face or Sigiriya. (**Medium Priority**)

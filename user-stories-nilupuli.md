# Passenger
 
 - As a passenger, I need to view the live location of buses on a map so I can minimize my wait time at the stop.
 - As a passenger, I need to see a digital layout of the bus interior with color-coded seat availability
 - As a passenger, I need to purchase a digital ticket and receive an instant receipt in my transaction history.
 - As a passenger, I need to report issues and attach a photo as evidence via the system.
 - As a passenger with disabilities, I need to see if the arriving bus is ramp-equipped so I can board safely.
 - As a passenger, I need to receive about traffic congestion or weather-related delays affecting specific route.

 # Admin

 - As an admin, I need a searchable database of all registered passengers, including their contact info and ride history.
 - As an admin, I need to view a real-time dashboard showing the status (Active, Idle, Maintenance) of every bus in the fleet.
 - As an admin, I need to track the ratio of "Tickets Purchased" vs. "Actual Passengers" to identify potential fare evasion.
 - As an admin, I need to generate daily, weekly, and monthly revenue reports categorized by route and payment method.

 # Driver

 - As a driver, I need to view my assigned route and daily schedule to ensure on-time departures.
 - As a driver, I need to see a real-time list of upcoming stops with the estimated number of passengers waiting.
 - As a driver, I need a direct communication channel to the Admin for reporting road blocks or emergencies.

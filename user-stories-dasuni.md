# User Stories for Metro Bus Project Sri Lanka

**Name:** Dasuni Nagodavithana  
**Date:** 2026-03-09
**Branch:** `doc/dasuni-research`

---
## 1. Passenger

- As a passenger I need to view available bus routes and schedules in the system
- As a passenger I should be able to track the real-time location of buses on a map
- As a passenger I should be able to buy or reserve a bus ticket through the system

---

## 2. Bus Driver

- As a bus driver I need to view my assigned route and daily schedule
- As a bus driver I should be able to update the bus status such as arrived, delayed, or in service
- As a bus driver I should be able to report traffic issues or technical problems through the system

---

## 3. Admin

- As an admin I need to manage bus routes and schedules
- As an admin I need to view details of buses and drivers in the system
- As an admin I should be able to view payment summaries and passenger statistics

## 2. Key Features Observed

### 🔵 India – Delhi Metro Rail
![Delhi Metro Rail Homepage](./screenshots/Delhi%20Metro%20Rail.png)
*Screenshot taken: 2025-03-08*

**Features noticed:**
- Interactive metro route map with station details
- Real time train information and service updates
- Fare calculator
- Station information (facilities, gates, nearby places)
- Multilingual support (English, Hindi)

**My observation:** The Delhi Metro website focuses strongly on providing practical travel information for daily commuters. The journey planner allows users to select a starting station and destination, after which the system displays the route, number of stations, travel time, and fare.

---

### 🔴 London – Transport for London (TfL)
![TfL Homepage](./screenshots/London.png)
*Screenshot taken: 2025-03-01*

**Features noticed:**
- Journey planner to find the best route between two locations in   London
- Real-time service status for Tube, buses, trains, and other transport services
- Contactless and Oyster card management, including viewing journey history and topping up cards online
- Account system to manage travel payments and refunds
- Disruption alerts banner at the top

**My observation:** The TfL website is designed as a complete transport information platform for London, rather than focusing only on metro services. The journey planner is a central feature that helps users quickly identify routes across multiple transport modes.

---

### 🟠 Hong Kong MTR
![Hong Kong MTR Homepage](./screenshots/Hong%20Kong.png)
*Screenshot taken: 2025-03-08*

**Features noticed:**
- Journey planner to find routes between MTR stations
- Interactive metro system map with all lines and stations
- Real time service updates and train information
- Fare information and fare saver details
- Mobile app promotion (MTR Mobile) for travel planning and service alerts

**My observation:** The MTR website provides clear and organized information for both daily commuters and tourists. The journey planner and system map make it easy to understand routes across the metro network.

## 5. My Personal Opinion


I think Sri Lanka’s metro website should focus on simplicity and easy access to information, since most users will want to quickly find routes, fares, and train timings. The website should have a clear layout with essential features such as a journey planner, fare calculator, and service updates, so that passengers can plan their trips easily.

Among the examples studied, the Singapore system provides a good reference because it presents important travel information in a simple and user-friendly way without making the interface complicated.

Another important aspect for Sri Lanka would be multilingual support, as passengers speak different languages. Providing the website in Sinhala, Tamil, and English would help ensure that all users can access and understand the information comfortably.
